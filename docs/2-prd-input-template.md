- [x] Node.js (Express)

**추가 요구사항**:
**답변**:

- Express.js 프레임워크

**선택지**:

- [x] PostgreSQL

**추가 요구사항**:
**답변**:

- PostgreSQL 14 이상

- 관계형 데이터 구조 활용
- 개발 환경: Docker Compose로 로컬 PostgreSQL 실행
- 프로덕션: Vercel Postgres 또는 Supabase

### 배포 환경

**선택지**:

- [x] Vercel

**추가 요구사항**:
**답변**:

- 프론트엔드: Vercel (자동 CI/CD)
- 백엔드 API: Vercel Serverless Functions
- 데이터베이스: Vercel Postgres (무료 티어 활용)
- 도메인: Vercel 제공 기본 도메인 사용 (예: cwh-todolist.vercel.app)
- 환경 변수: Vercel 대시보드에서 관리
- Git 기반 자동 배포 (main 브랜치 푸시 시)
