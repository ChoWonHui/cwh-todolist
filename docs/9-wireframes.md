## 1. 와이어프레임 개요

### 1.1 설계 원칙

- **단순하고 깔끔함 (Simple & Clean)**: 불필요한 요소 제거, 여백 충분히 활용
- **빠른 입력 (Fast Input)**: 최소 클릭/탭으로 할일 추가
- **명확한 상태 표시 (Clear Status)**: 할일 상태와 마감일 한눈에 확인
- **익숙한 인터페이스 (Familiar)**: 네이버 캘린더와 유사한 레이아웃

### 1.2 색상 팔레트

#### 라이트 모드
- **Primary Blue**: `#5B76F7` (버튼, 링크, 선택된 항목)
- **Primary Hover**: `#4A62E6`
- **Red**: `#FF4D4D` (기한 초과, 삭제)
- **Green**: `#00C73C` (완료)
- **Orange**: `#FF8C42` (마감 임박)
- **White**: `#FFFFFF` (배경)
- **Gray-50**: `#F9FAFB` (사이드바 배경)
- **Gray-100**: `#F3F4F6` (카드 배경)
- **Gray-600**: `#666666` (보조 텍스트)
- **Gray-900**: `#222222` (주요 텍스트)

#### 다크 모드
- **Dark Background**: `#121212` (메인 배경)
- **Dark Surface**: `#1E1E1E` (카드, 폼 배경)
- **Dark Surface Secondary**: `#2D2D2D` (사이드바 배경)
- **Dark Text Primary**: `#FFFFFF` (주요 텍스트)
- **Dark Text Secondary**: `#B3B3B3` (보조 텍스트)
- **Dark Text Tertiary**: `#808080` (세컨더리 텍스트)
- **Primary Blue Dark**: `#638AFF` (다크 모드용 버튼, 링크)
- **Primary Hover Dark**: `#7599FF`

### 1.3 폰트 패밀리 및 크기

- **폰트 패밀리**: `-apple-system, BlinkMacSystemFont, "Malgun Gothic", "맑은 고딕", sans-serif`

- **부제목 (H2)**: 20px, 600
- **본문**: 14px, 400
- **작은 텍스트**: 12px, 400

## 4. 상세 와이어프레임

### 4.1 WF-001: 회원가입 화면

- **화면 구조**:
  ```
  ┌─────────────────────────────────────────────────────┐
  │  Header (로고만 표시, 로그인 링크)                    │
  ├─────────────────────────────────────────────────────┤
  │                                                     │
  │  [cwh-todolist]                                     │
  │  ┌─────────────────────────────────────────────────┐ │
  │  │                                                 │ │
  │  │  사용자명 입력 필드                              │ │
  │  │                                                 │ │
  │  │  이메일 입력 필드                                │ │
  │  │                                                 │ │
  │  │  비밀번호 입력 필드                              │ │
  │  │                                                 │ │
  │  │  [회원가입] 버튼                                 │ │
  │  │                                                 │ │
  │  └─────────────────────────────────────────────────┘ │
  │                                                     │
  │  "이미 계정이 있나요? [로그인]" 링크                 │
  │                                                     │
  └─────────────────────────────────────────────────────┘
  ```

- **UI 구성**:
  - 상단: 로고와 로그인 링크
  - 중앙: 폼 컨테이너 (사용자명, 이메일, 비밀번호 필드)
  - 하단: 회원가입 버튼과 로그인 링크
- **기능**:
  - 폼 유효성 검사 (이메일 형식, 비밀번호 최소 8자)
  - 중복 이메일 확인
  - 회원가입 성공 시 자동 로그인 및 할일 목록 페이지 이동

### 4.2 WF-002: 로그인 화면

- **화면 구조**:
  ```
  ┌─────────────────────────────────────────────────────┐
  │  Header (로고만 표시, 회원가입 링크)                  │
  ├─────────────────────────────────────────────────────┤
  │                                                     │
  │  [cwh-todolist]                                     │
  │  ┌─────────────────────────────────────────────────┐ │
  │  │                                                 │ │
  │  │  이메일 입력 필드                                │ │
  │  │                                                 │ │
  │  │  비밀번호 입력 필드                              │ │
  │  │                                                 │ │
  │  │  [로그인] 버튼                                   │ │
  │  │                                                 │ │
  │  └─────────────────────────────────────────────────┘ │
  │                                                     │
  │  "계정이 없나요? [회원가입]" 링크                   │
  │                                                     │
  └─────────────────────────────────────────────────────┘
  ```

- **UI 구성**:
  - 상단: 로고와 회원가입 링크
  - 중앙: 폼 컨테이너 (이메일, 비밀번호 필드)
  - 하단: 로그인 버튼과 회원가입 링크

  - 로그인 성공 시 할일 목록 페이지 이동
  - 로그인 실패 시 에러 메시지 표시

### 4.3 WF-003: 할일 목록 화면

- **화면 구조**:
  ```
  ┌─────────────────────────────────────────────────────┐
  │  Header (로고, 사용자명, 로그아웃)                    │
  ├──────────┬──────────────────────────────────────────┤
  │          │                                          │
  │ Sidebar  │  Main Content (할일 목록)                │
  │          │                                          │
  │ • 할일   │  ┌─────────────────────────────────┐    │
  │   추가   │  │ [할일 1]                        │    │
  │          │  │ 제목: 프로젝트 회의 준비        │    │
  │ • 전체   │  │ 마감: 2025-11-28 (D-2)         │    │
  │   할일   │  │ [수정] [삭제]                  │    │
  │          │  └─────────────────────────────────┘    │
  │ • 오늘   │                                          │
  │   할일   │  ┌─────────────────────────────────┐    │
  │          │  │ [할일 2]                        │    │
  │          │  │ ...                             │    │
  │          │  └─────────────────────────────────┘    │
  │          │                                          │
  │          │  ┌─────────────────────────────────┐    │
  │          │  │ [할일 3 - 공통 일정]           │    │
  │          │  │ 제목: 광복절                    │    │
  │          │  │ 날짜: 2025-08-15                │    │
  │          │  │ [삭제 불가]                     │    │
  │          │  └─────────────────────────────────┘    │
  │          │                                          │
  └──────────┴──────────────────────────────────────────┘
  ```

- **UI 구성**:
  - 상단 헤더: 로고, 사용자명, 로그아웃 버튼
  - 좌측 사이드바: 할일 추가, 필터링 메뉴
  - 우측 메인: 할일 카드 목록
- **기능**:
  - 할일 목록 표시 (제목, 마감일, 상태)
  - "할일 추가" 버튼 클릭 시 모달 팝업
  - 할일 카드 수정/삭제 기능
  - 공통 일정(국경일) 표시 (삭제 불가 표시)
  - 마감일이 임박한 할일 강조

  │  │                                                 │ │
  │  └─────────────────────────────────────────────────┘ │
  │                                                     │
  │  시작일                                              │
  │  ┌─────────────────────────────────────────────────┐ │
  │  │                                                 │ │
  │  │  날짜 피커                                       │ │
  │  │                                                 │ │
  │  └─────────────────────────────────────────────────┘ │
  │                                                     │
  │  마감일                                              │
  │  ┌─────────────────────────────────────────────────┐ │
  │  │                                                 │ │
  │  │  날짜 피커                                       │ │
  │  │                                                 │ │
  │  └─────────────────────────────────────────────────┘ │

- **UI 구성**:
  - 상단: 제목 (할일 추가/수정)
  - 중앙: 폼 필드 (제목, 설명, 시작일, 마감일)
  - 하단: 저장/취소 버튼
- **기능**:
  - 제목 필수 입력 검사
  - 시작일 < 마감일 유효성 검사
  - 저장 후 모달 닫히고 할일 목록 갱신

## 5. 반응형 가이드

### 5.1 데스크톱 (1024px 이상)

- 사이드바 고정, 메인 콘텐츠 영역 확장
- 할일 카드를 그리드 형식으로 3~4열 표시
- 전체 화면 모달 사용

### 5.2 태블릿 (768px ~ 1023px)

- 사이드바 접기/펼치기 기능
- 할일 카드를 그리드 형식으로 2열 표시
- 모달은 중간 크기 유지

### 5.3 모바일 (767px 이하)

- 사이드바 하단 탭으로 전환
- 할일 카드를 리스트 형식으로 1열 표시
- 모달은 전체 화면으로 표시
